Fullstack API TESTE

Esta é uma aplicação web construída com Angular e Spring Boot, permitindo que você gerencie contas de usuários em um sistema.

  • Criar Usuário - Adicione novo usuário.
  • Listar Usuários - Listar todos os usuários.
  • Editar Usuário (Acessível a partir da Lista de Usuários) - Modificar um usuário.
  • Deletar Usuário (Acessível a partir da Lista de Usuários) - Deletar um usuário.

Para visualizar os endpoints da API, consulte a documentação do Swagger: Swagger API Docs
